#  Space Mission Control Dashboard

A real-time full-stack web application designed for monitoring and managing space missions, satellites, and live telemetry systems. The platform provides interactive dashboards, mission tracking, ISS monitoring, and real-time data visualization using modern web technologies.

---

## Features

*  Mission management system
*  Real-time telemetry monitoring
*  Satellite tracking dashboard
*  ISS live position tracker
*  Interactive charts and analytics
*  Mission logs with timestamps
*  WebSocket live updates
*  SQLite database integration
*  Responsive UI with Tailwind CSS
*  Fast frontend using React + Vite

---

##  Tech Stack

| Technology   | Usage                   |
| ------------ | ----------------------- |
| React + Vite | Frontend                |
| Tailwind CSS | UI Styling              |
| Node.js      | Backend Runtime         |
| Express.js   | API Server              |
| Socket.io    | Real-time Communication |
| SQLite       | Database                |

##  Installation

```bash
# Install dependencies
npm install

# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install
```

---

##  Run the Project

```bash
# Run frontend and backend together
npm run dev
```

Frontend:

```bash
http://localhost:5173
```

Backend:

```bash
http://localhost:3001
```
